# One Hoster QA Automator

An automated website auditing tool designed for One Hoster web agency.

## Architecture
- Frontend: React 19 + Vite (TypeScript + Tailwind CSS)
- Backend: Node.js + Express
- Audit Engine: Google Lighthouse (Node CLI), Cheerio (Scraping)
- Database: SQLite (via Prisma)

## Setup
1. `npm install`
2. `npx prisma init`
3. `npm run dev`

## Features
- SEO Validation
- Performance (Lighthouse)
- Broken Link Checking
- Image Optimization Scanning
